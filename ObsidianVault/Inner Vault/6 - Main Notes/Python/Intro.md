
Ao trabalhar com Python, temos sempre um arquivo de texto chamado `requirements.txt`. Esse arquivo serve para indicar quais são as bibliotecas que iremos usar nos projetos.

Para criar e instalar um ambiente novo do 0 em python, basta usar os comandos

- `python3 -m venv venv` -> inicializa um diretório chamado venv.
- `source venv/bin/activate` -> entra no ambiente para poder instalar as dependências.

Alguns comandos simples de bibliotecas:

`pip install` serve para instalar dependências no python
`pip freeze > requirements.txt` congela as dependências e joga em um arquivo de texto
