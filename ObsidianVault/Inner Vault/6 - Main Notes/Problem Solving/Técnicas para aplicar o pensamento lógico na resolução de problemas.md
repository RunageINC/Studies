
O pensamento lógico é fundamental para enfrentar desafios de maneira eficiente. Aplicando técnicas como a **quebra do problema em partes menores**, a **análise de causas e efeitos** para identificar suas origens, e o **raciocínio dedutivo e indutivo** para obter conclusões bem fundamentadas, é possível tratar questões complexas de forma mais **organizada e objetiva**.

Seguem algumas técnicas para aplicar o pensamento lógico:

## Árvores de Decisão

![[Screenshot 2025-11-20 at 15.22.04.png]]

As árvores de decisão são diagramas que ilustram os possíveis resultados decorrentes de uma sequência de escolhas.

Cada ramo da árvore simboliza uma decisão, e cada nó final corresponde a um resultado específico. Essa técnica é útil para:

- **Analisar opções**: comparar diferentes alternativas e seus potenciais resultados
- **Gerenciar riscos**: Identificar os riscos envolvidos em cada decisão
- **Aprimorar processos**: Determinar o caminho mais eficiente para atingir um objetivo

## Análise de Pareto

![[Pasted image 20251120152544.png]]

Também conhecida como Diagrama de Pareto ou Princípio 80/20, é uma ferramenta estatística que nos ajuda a identificar as causas mais significativas de um problema, permitindo que foquemos nossos esforços onde eles trarão o maior impacto.

Em termos simples, a regra 80/20 sugere que, em muitos casos, 80% dos resultados são provenientes de apenas 20% das causas. Ou seja, uma pequena porcentagem de fatores contribui de forma desproporcional para o resultado final.

- **Defina o problema**: Qual é o problema que você quer analisar? Seja específico.
- **Colete os dados**: Identifique os fatores que contribuem para o problema e colete dados sobre sua frequência ou impacto.
- **Classifique os dados**: Organize os dados em ordem decrescente de frequência ou impacto.
- **Crie o gráfico**: Construa um gráfico de barras, onde a altura de cada barra representa a frequência ou o impacto de cada fator.
- **Calcule as porcentagens**: Calcule a porcentagem acumulada de cada fator e trace uma linha para representar essa porcentagem.
- **Identifique os 20% mais importantes**: Os fatores que correspondem aos 20% mais significativos são os que devem ser priorizados.

## Diagrama de Fluxos

![[Pasted image 20251120153019.png]]

Os diagramas de fluxo representam os passos sequenciais de um processo. Eles são úteis para:

- **Visualizar processos**: Facilitar a compreensão de como um processo funciona;
- **Identificar gargalos**: Encontrar pontos de estrangulamento no processo;
- **Documentar processos**: Criar um registro visual do processo;

Ao desenvolver o hábito de pensar de forma lógica e sistemática, é possível aprimorar a capacidade de resolver problemas e tomar decisões mais assertivas.
