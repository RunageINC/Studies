Identifique o problema
  • Defina o problema de forma clara, evitando generalizações
  • Descreva o problema de maneira objetiva e direta
  • Identifique os sintomas e as possíveis causas
Analise o problema
  • Reúna informações improtantes sobre o problema
  • Identifique as causas principais utilizando ferramentas como o diagrama de Ishikawa (espinha de peixe)
  • Examine as possíveis consequências do problema
Gere alternativas
  • Brainstorming: produzir o máximo de ideias possíveis para resolver o problema
  • Análise das opções: considerar os prós e contras de cada solução
  • Escolha da melhor opção: selecionar a solução mais prática e eficiente
